== Description ==

A very simple yet useful plugin to show different content to registered and non-registered users using shortcodes.

It also provides the option to display a message or various content for non-registered users using native wordpress editor from plugin's info page.

Features include:

* Option page with informations on how to use the shortcode.
* Editor within option page to set the custom messages for non-registered users viewing page or post
* Preset options with basic usage. 5 Presets with different content


**Simplicity**
'Visibility Custom Content' is a user-friendly WordPress plugin that lets you display different content to registered and non-registered users using shortcodes. With this plugin, you can easily create content that only registered users can see, while non-registered users will view content that you define in the plugin settings. It's a great way to offer exclusive content to your site's members while still providing something engaging for non-members. You can also customize the look of your content with preset styles or your own CSS. Optional, some coding required if you want to design your own message using inline css!



== Installation ==

= Automatic installation =


 1. Log in to your WordPress dashboard.
 2. Navigate to the Plugins menu, and click "Add New".
 3. Search and locate 'Visibility Custom Content' plugin.
 4. Click 'Install Now', and WordPress will take it from there.


= Manual installation =

Manual installation method requires downloading the 'Visibility Custom Content' plugin and uploading it to your web server via your FTP application. The WordPress codex contains instructions on [how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
